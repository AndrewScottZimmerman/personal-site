# Design
## Feeling and Vibe
Clean, classic, minimal, professional. Personal photography used to accent pieces. Gentleman vibe, but not sterile.

## Typography
Headings: Source Serif 4, h1 and h2 weight of 600, h3 weight of 500
Body: Source Serif 4, weight of 400-500 leading of 1.6-1.7
UI, Nav, and Labels: Inter, weight of 400-500
Code and Tech Labels: JetBrains Mono, weight of 400-500.

## Type Scale
method: rem
root: 16px
h1: 3rem (48px) - hero title
h2: 2rem (32px) - page titles
h3: 1.5rem (24px) - section headings
body: 1.0625rem (17px) - prose body
small: 0.875rem (14px) - captions, footer dislaimer
micro: 0.75rem (12px) - metadata, tech tags, etc.

## Color Pallette
Background: #FAF7F2
Text Primary: #1F1F1F
Text Secondary: #5A5A5A
Accent: #1B2A4E
Accent Muted: #E8E4DC

## Spacing and White Space
Spacing Scale: 4px base, multiples afterwards.

Hero: Full Width
